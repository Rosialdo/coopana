## How to run

### ENV

- Create user in MySQL:

```mysql
CREATE USER 'MYSQL_USER'@'MYSQL_HOST' IDENTIFIED BY 'MYSQL_PASSWORD';
```

- Create .env file:

`touch .env`

- Follow `.env.example`template, adding the values to the fields:

```env
MYSQL_USER=
MYSQL_HOST=
MYSQL_PASSWORD=;
```
